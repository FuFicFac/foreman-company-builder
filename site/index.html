Foreman

Paperclip is the company. Foreman runs the crew.

curl -fsSL https://get.foreman.dev | zsh One command. Discovers your AI tools. Assigns roles. Starts dispatching.

What it does

  • Discovers what AI tools you have (Cursor, Claude, Codex, Ollama, whatever)
  • Assigns roles — inspector (strongest), builder (fastest), cheap (classification)
  • Dispatches builders to do the work
  • Dispatches inspectors to review the work
  • Arbitrates — accept, reject, or send back for fixes
  • Escalates after 3 failed attempts (no infinite retry spirals)

Built for

Software

Dispatch code. Inspect for bugs. Verify builds.

Creative Writing

Dispatch chapters. Six editors review independently.

YouTube

Dispatch scripts. Inspect hooks, pacing, retention.

Marketing

Dispatch copy. Inspect brand fit, clarity, conversion.

How it works

1. Init

foreman init — discovers your fleet, detects API keys, picks a brain

2. Dispatch

foreman dispatch — sends work to the right agent for the job

3. Inspect

An independent reviewer checks the work before it ships

4. Accept or fix

Good enough → ship. Problems → specific feedback → one more try

5. Escalate

Three strikes and the problem goes up the chain. No spirals.

Modules

  • Four built-in swarm templates: software, creative writing, YouTube, marketing
  • Install community modules from GitHub
  • Write your own — a module is just a folder with module.json and SKILL.md
  • Foreman notifies you when modules have updates
